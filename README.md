This project uses a Residual U-Net architecutre to attempt using stem separation into
Drums, Vocals, Bass, and Other.

This was a first attempt at creating a model so results were sub par and the nature of stem separation is one of the harder 
models to create especially with not having a gpu with a lot of VRAM.

Gemini aided creating this ai training and model.

Got to experience tuning AI model.
